## Lab Questions

### In plain English describe the function and use of a multiplexer.
A multiplexer is a combination of logic circuits that use multiple input signals and sends it to a singal output. The input that goes to the output is determined by specific lines. Multiplexers are usually used so that multiple signals are able to share a single output.

### In plain English describe the function and use of a demultiplexer.
A demultiplexer is a logic curcuit that takes a single input and routes it to one of several outputs. The output that receives the signal depends on the selected lines. Demultiplexers are use when a signal needs to be distributed to multiple outputs.

### What other uses might these circuits have? (Think Shannon’s)
Multiplexers can also be implemented using boolean functions using Shannon's expansion theorm. Using specific lines as variables, it allows for different inputs to be represented in a logic function. Because of this, multiplexers can act as general logic functions which allows for complex logic circuits to be built using multiplexers.
